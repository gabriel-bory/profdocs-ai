# Documentation Site

ProfDocs AI includes a dedicated documentation site built with Astro Starlight.

## Location

```text
apps/docs
```

## Purpose

The documentation site provides a professional, browsable version of the project's documentation as code.

It includes:

- Product vision
- Architecture
- Roadmap
- Database model
- Development workflow
- Package management
- Full-stack foundation
- ADRs

## Technology

- Astro
- Starlight
- Markdown / MDX
- Built-in light and dark mode

## Commands

From the repository root:

```bash
pnpm dev:docs
pnpm build:docs
```
