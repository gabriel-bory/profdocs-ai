# Roadmap

## MVP 0.1 — Project Foundation

Goal: create a professional full-stack foundation.

Includes:

- Public repository
- README
- Documentation as code
- Monorepo structure
- Angular 21 frontend foundation
- NestJS app
- Docker Compose with PostgreSQL 18
- Initial dashboard shell
- Environment example file
- Initial architecture documentation

## MVP 0.2 — Simple Document Ingestion

Goal: process simple text-based documents.

Includes:

- TXT document support
- Markdown document support
- Document metadata
- Content extraction
- Chunking
- Document status
- Content preview

Document status values:

- uploaded
- processing
- ready
- failed

## MVP 0.3 — Minimal RAG

Goal: implement the first AI-powered workflow.

Includes:

- Embeddings
- pgvector
- Semantic search
- Ask endpoint
- Source fragments
- AiProvider abstraction
- Basic response grounded in retrieved chunks

## MVP 0.4 — Portfolio-Ready Product

Goal: prepare the project for GitHub and LinkedIn presentation.

Includes:

- Improved chat UI
- Conversation history
- PDF support
- Screenshots
- Demo documentation
- Visual polish
- README improvements
- LinkedIn project description
