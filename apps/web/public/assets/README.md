# ProfDocs AI web assets

Static assets placed here are copied by Angular from `apps/web/public`.

Asset strategy for poor network conditions:

- Keep critical UI independent from remote fonts or icon CDNs.
- Use system fonts by default.
- Use local SVGs for icons when custom icons become necessary.
- Keep images optimized and optional for the SaaS dashboard.
- Keep Gabriel Bory Prevez's professional photo for future About/Author sections, not the main app shell.
