## Status

Accepted

## Context

ProfDocs AI needs to demonstrate a professional full-stack architecture aligned with Gabriel Bory Prevez's profile.

## Decision

Use:

- Angular 21 for the frontend.
- NestJS for the backend.
- PostgreSQL 18 for the database.
- pgvector in a later RAG phase.

## Consequences

- Strong TypeScript alignment.
- Professional backend architecture.
- Clear path toward RAG with PostgreSQL and pgvector.
