# Database Model

The database model evolves gradually across MVP versions.

## MVP 0.1

Conceptual entities only:

- Workspace
- Document

## MVP 0.2

Adds:

- DocumentChunk
- Document status lifecycle

## MVP 0.3

Adds:

- Embeddings
- Semantic search
- RAG query logs

## MVP 0.4

Adds:

- Conversation
- Message
- Source references
