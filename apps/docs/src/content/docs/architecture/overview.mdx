# Architecture Overview

ProfDocs AI follows a modular full-stack architecture.

```text
Angular Web App
    |
    v
NestJS API
    |
    v
PostgreSQL 18
```

The documentation site is generated separately with Astro Starlight:

```text
Astro Starlight Docs
    |
    v
Static documentation site
```

## Principles

- Start small.
- Keep modules clear.
- Document decisions.
- Avoid premature complexity.
- Keep AI provider integration abstract.
- Build incrementally.
